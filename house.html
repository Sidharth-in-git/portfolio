House Price Predictor

Regression models & interpretability for housing price estimation

Overview

This project models house prices using features such as area, number of rooms, location, age, and amenities. The workflow includes EDA, feature engineering (location encoding, interaction terms), model selection (linear models, RandomForest, XGBoost), and interpretation using SHAP values.

Key points:

  • Feature engineering for location and non-linear effects.
  • Robust evaluation using cross-validation and hold-out sets.
  • Interpretability: SHAP-based explanations to communicate feature contributions.

Tech: Python, Pandas, Scikit-learn, XGBoost, SHAP
